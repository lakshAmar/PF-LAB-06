## Lab Objectives
The objective of this lab is to understand and implement different looping structures in 
C programming, including for loop, while loop, and do-while loop, to solve real-world 
problems such as attendance systems, ATM simulation, password verification, data analysis,
and reservation systems. This lab helps in developing logical thinking and practical 
problem-solving skills using iterative control structures.
